The Islands

Indonesia is an archipelago country that has more than 17.000++ islands and 62% of the sea area

There are 7 main islands, Sumatra, Kalimantan, Jawa, Sulawesi, Papua Barat, Kepulauan Maluku, Kepulauan Suanda Kecil
